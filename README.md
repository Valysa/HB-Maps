# HB-Maps
Un repo qui contiendra les différentes versions des maps faites sur HB

Le but de ce repo est de garder à un seul endroit et proprement, un historique des différentes cartes d'HB.
Je fais les maps de la façon suivante

J'utilise le mod "map.png"
Je passe sur chaque carte d'une grande map et je la save avec F8
Je crée un nouveau fichier sur GIMP de la taille de Nb de carte en largeur * taille d'une carte
Je met la grille à la taille d'une carte
Je coche dans "views" snap to grid et show grid
Je drag and drop les maps une par une et je reconstitue la carte comme un puzzle

Dans ce repo, chaque map aura son dossier. Le dossier contiens le fichier Gimp, le png de la map "vierge", et éventuellement la map annotée si j'en fais une.

Chaque changement d'une carte entrainera un commit, ce qui permettra de garder une trace des cartes les plus anciennes.
(Les noms du commits sont toujours [NomDeLaCarte] [Création/Modification] [Le numéro des cartes changées] et les zones RP changées éventuellement)
